## How to use a Overlay/Tint

- Open Helperbird on Chrome, Firefox or Edge
- Scroll to the **Overlay** card.
- Toggle the on/off switch to **On**.
- A overlay, will now appear on the page.
- Thats it.

## Features

- Overlay/Tint color
- Overlay/Tint opacity
