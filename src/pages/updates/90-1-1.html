Updates: All new ways to learn

Whats new?

  • 📗 There is now a read it later button in the quick access menu.
  • 🎨 The highlight menu is on by default. You can always turn if off.
  • 🗣️ Text to speech is now quicker and better.
  • ▶️ Removed the pause feature, so on you can turn Helperbird on or off. Making things easier.
  • 👤 Added onboarding/welcome message to all new users.
  • If there were no favorite features, there is now a default card.
  • Text to speech no longer stops after 15 seconds on Windows.
